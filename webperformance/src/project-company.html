This was for a database class to demo connecting to a database with PHP. pretty easy stuff!

You can check out the project at company search. Try searching for "Smith".

This project connects to the database with PHP. You should be able to search the database by last name and return the result of searching an employee table. It was good to learn how PHP works and how, to avoid SQL injection by using prepared statements. The other benefit of prepared statements is precompiled queries that only need to be parsed one time, but executed many times.
